Notebooks should be put into the right folder based on the final goals. There are several different goals for a notebook

1. Data: this is purely about data parsing and cleaning parts, and to save it to local folders. It should be in general and has little correlation with how the data is used in future when explore patterns. 
1. Pattern Exploration: this is more about assumption tests without any known results. It is pretty an early stage of a research may end or not end with a strategy. 
1. Model Documentations, this is more about mature ideas and try to documents and show the evidences so that we can follow the steps to replicate and apply it to real time data
1. Real time monitors: this is for notebooks used on real time market analysis and predictions, can be used for an investment guidances. All the methodology in this folder should be documented well in model documentations folder
1. misc: this folder for notebook has no clear purposes yet.
1. lib functions: this folder is for function usage illustration.
